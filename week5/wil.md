5주차에서는 react 에서 외부 서버의 api를 호출하고 데이터를 화면에 렌더링하는 과정을 알게되었다. 클라이언트가 서버에 데이터를 요청하면 서버가 응답을 보내며, api는 서로 다른 두 소프트웨어 간의 상호작용을 돕는 인터페이스 역할을 한다는 점을 알게 되었다. 또한 잦은 DOM 조작은 브라우저 리소스를 많이 소모하므로, react는 메모리상에 가상의 DOM을 만들어 조작한 뒤 변경된 결과를 일괄적으로 실제 DOM에 반영한다는 개념을 알게 되었다. 그리고 실습 과정에서 외부 시스템과 안전하게 동기화하기 위해 useEffect 훅을 사용해봤다. 마지막으로 코드를 컴포넌트로 분리하고 props를 활용해 하위 컴포넌트로 속성 값을 전달하는 리팩토링도 실습해봤다.
