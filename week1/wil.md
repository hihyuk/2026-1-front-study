1주차에서 html과 javascript 에 대해 공부했다.
vs 코드 프로그램으로 html 파일을 만들고 live server extension으로 실행했다.
html은 hypertext markup language로 팀 버너스리가 인터넷 상에서 정보의 검색 및 교환을 용이하게 하기 위해 통일된 양식을 만든 것이다.
markup language란 tag를 사용해서 구조적인 문서를 나타내는 언어이고
hypertext는 링크를 사용하여 페이지를 연결하는 것이다.
대표적인 태그에는 head, title, body, h1, p, a 태그 등이 있다.
script 태그를 사용하여 console.log로 콘솔창에 문자열을 출력하고
alert로 알림창을 출력하는 방법을 공부했다.
script는 멈춰있는 문서를 동작하게 하기 위해 필요하다.
