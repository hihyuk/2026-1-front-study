2주차에서는 html 파일에서 js 파일을 마이그레이션하여 문서를 동적으로 만드는 방법을 공부했다.
html 파일 내 script 태그 안에 코드를 작성하게 되면 문서가 길어지고 html의 문서라는 본질을 해치게 된다.
그래서 html과 javascript 코드를 따로 분리해야 한다는 생각을 가지게 되고 html파일에서 `<script src="./app.js"></script>`의 방식으로 js 코드를 분리한다.
이때 위 코드를 `</body>` 앞에 작성해야 되는데 이는 브라우저의 동작 방식과 관련이 있다.
브라우저가 html 파일을 열면 위에서부터 한줄씩 읽게 되는데 중간에 script를 만나면 파싱을 중단하고 js 파일을 다운받아 실행하게 된다. 이 때 두가지 문제가 발생하는데 html 해석이 늦어지고 순서가 꼬이게 될 수 있다.



변수 선언: 데이터가 계속 변할 수 있는 값이면 let을, 변하지 않는 상수면 const를 사용한다.  
연산: 기본적인 사칙연산 외에도 값을 1씩 더하거나 빼는 증감연산자(++, --), 값을 누적하는 복합할당 연산자(+=, -=)를 사용할 수 있다.  
반복문: for문, while문 을 동일한 로직을 여러 번 실행할 때 사용한다.  
조건문: if문, switch문 으로 특정 조건이 참인지 거짓인지에 따라 로직을 분기할 때 사용한다.  
함수: 특정 작업을 반복 수행할 때 재사용할 수 있게 만든 코드 블록이다.
