4주차에서 react의 탄생배경과 동작 원리를 공부했다. 기존의 빈번하고 무거운 DOM 조작 문제를 해결하기 위해 React는 메모리상에 Virtual DOM을 생성하고 변경 사항을 비교하여 실제 DOM에 일괄적으로 반영하여 렌더링 성능을 최적화한다는 것을 알게 되었다. 실습에서 전체 틀, 버튼, 숫자 부분으로 컴포넌트를 분리해서 재사용성을 높였고 이 과정에서 상위 컴포넌트의 상태(state)를 하위 컴포넌트에게 props로 전달해 컴포넌트 간의 관계를 구현하여 데이터 흐름을 이해할 수 있었다. 또한, 컴포넌트는 Mount, Update, Unmount의 3단계로 구분되며 새로운 props나 state를 수신할때 컴포넌트가 update 된다는 것을 알게 되었다.
